# SuperChat - Real-time Chat Application

SuperChat is a real-time chat application built with React and Firebase. It allows users to sign in with their Google account and chat with others in a shared chat room.

## Features
- Real-time messaging with Firebase Firestore
- Google Authentication
- Responsive and modern UI
- Easy deployment

## Technologies Used
- React.js
- Firebase Authentication
- Firebase Firestore
- React Firebase Hooks

## Getting Started

### Prerequisites
- Node.js and npm installed
- A Firebase project (for your own deployment)
